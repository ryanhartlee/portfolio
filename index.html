Full stack application that allows users to post and view secret menu items from various restaurants. Built using the MERN stack, React + Redux was used on the front end, while Node and Express runs the backend. Mongo + Mongoose stores both user and recipe information.

App that allows users to create an account and track expenses. User data is securely stored in a database using the 'Passport' node package. Users can add expenses and filter them by category and date, as well as view a graph breakdown of their spend demographics. Built using Node, Express.js, Sequelize/MySQL, and HTML5/CSS.
